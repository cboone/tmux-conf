# Colour Options

> `tmux -L unconfigured -f /dev/null start-server \; show-options -g \; show-options -gw | grep colour`

## Display

```tmux
display-panes-active-colour red
display-panes-colour blue
```

```tmux
cursor-colour none
clock-mode-colour blue
```

## Pane

```tmux
pane-border-indicators colour
pane-colours
```
